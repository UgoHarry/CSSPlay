CSS Play - Variables

So the idea behind this is very simple

  • Global styles are created using CSS variables
  • These global styles are applied to elements of your choosing
  • JavaScript uses certain input elements to adjust/changes these global styles
  • Changes on the global styles reflect on applied elements, in real time
